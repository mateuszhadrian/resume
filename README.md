## Personal Projects

### GeoCanvas - Browser-Based Geometric Shape Editor

*Next.js, React, TypeScript, Konva.js, Zustand + Immer, Tailwind CSS v4, Jest, Playwright*
GitHub: [github.com/mateuszhadrian/geo-canvas-app](https://github.com/mateuszhadrian/geo-canvas-app)

Minimalist canvas editor inspired by Excalidraw and tldraw - with multi-layer support, marquee multi-select, and full undo/redo via a custom Command Pattern history. Shapes are self-describing modules registered in a central shape registry, keeping the rest of the app shape-agnostic.

**Key takeaways:** non-trivial frontend architecture on canvas (Konva.js), serializable state design (CanvasDocument format with schema versioning), custom Command Pattern history, and CI/CD pipeline with unit, component, and E2E tests on every push.

### 10x Cards - AI-Powered Flashcard Generation Web App

*Astro, React, TypeScript, Supabase (PostgreSQL + RLS), Cloudflare Pages, Vitest, Playwright*
Live: [10x-cards-3k2.pages.dev](https://10x-cards-3k2.pages.dev) | GitHub: [github.com/mateuszhadrian/10x-cards](https://github.com/mateuszhadrian/10x-cards)

Generates up to 30 flashcards from user-provided text (1,000-10,000 characters) using AI. Features Supabase Auth, multi-layer validation (UI/API/RLS), and a review flow allowing users to accept, edit, or reject each card before saving.
